Instruments for pedaludic cartography — field recording, live tracking, data processing and route analysis.

Real-time GPS tracking with live trail, environmental data, PDIs and Telegram integration. Multiple riders visible simultaneously on the same map.

Cropear
GPX · GeoJSON · Bounding Box

Clip GPX or GeoJSON routes to a defined bounding box. Set center coordinates and delta, filter by point count, export as JSON, GeoJSON or p5.js sketch.

Rebase multiple GPX tracks to a shared origin point. Generates a bouquet-like multi-track GPX where all rides start from the same coordinate.
